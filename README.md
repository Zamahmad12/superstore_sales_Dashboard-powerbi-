This project is an interactive Power BI dashboard built to analyze Superstore sales data and extract meaningful business insights.

What this dashboard shows:

Total Sales, Profit, and Quantity.
Sales analysis by Category & Sub-Category.
Segment-wise sales and profit comparison.
State-wise sales performance.
Monthly sales trends across years.
Interactive filters for Region and Payment Mode.

Tools Used:

Power BI.
Data Cleaning & Modeling.
DAX for calculations.

Key Objective:

To convert raw sales data into a visually clear and interactive dashboard that helps understand business performance at a glance.
